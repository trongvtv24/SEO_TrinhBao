## 5) Khung bài viết (Outline chuẩn bài commercial)

## H1: [Keyword + lợi ích quyết định]
- Ví dụ: **Nên chọn camera gia đình loại nào? So sánh theo nhu cầu và ngân sách**

## Mở bài
1. Xác nhận người đọc đang phân vân
2. Nêu tiêu chí so sánh sẽ dùng
3. Hứa giúp chọn nhanh đúng nhu cầu

## H2. Trả lời nhanh (Quick recommendation)
- Nếu bạn cần [nhu cầu A] → chọn [gợi ý]
- Nếu bạn cần [nhu cầu B] → chọn [gợi ý]
- Nếu ngân sách [X] → ưu tiên [gợi ý]

## H2. Tiêu chí chọn mua / so sánh

- CTA rõ: xem mẫu / nhận tư vấn / báo giá / đặt lịch

## 6) Hướng dẫn viết (cho writer/AI)

### Nguyên tắc quan trọng
- Không viết chung chung kiểu “loại nào cũng tốt”
- Phải có **khuyến nghị cụ thể theo tình huống**
- Có **bảng so sánh** (rất quan trọng)
- Có **ưu/nhược điểm** (tăng độ tin cậy)
- Nếu có giá: ghi rõ **giá tham khảo** + điều kiện thay đổi
- Ưu tiên ngôn ngữ giúp ra quyết định

### Công thức viết từng mục so sánh
1. Nêu tiêu chí
2. So sánh nhanh
3. Kết luận ai nên chọn cái nào
4. Chèn link sang trang đích phù hợp

---

## 7) SEO On-page Checklist trước publish

### Nội dung
- [ ] Trả lời đúng intent “so sánh/chọn mua”
- [ ] Có bảng so sánh
- [ ] Có khuyến nghị theo nhu cầu/ngân sách
- [ ] Có ưu/nhược điểm
- [ ] Có CTA rõ + internal link về trang chốt chuyển đổi

### Heading / Metadata
- [ ] 1 H1 duy nhất
- [ ] Title chứa keyword + lợi ích ra quyết định
- [ ] Meta mô tả nêu rõ tiêu chí so sánh / đối tượng phù hợp
- [ ] Slug ngắn gọn

### Ảnh / Media
- [ ] Ảnh sản phẩm minh họa đúng nội dung
- [ ] ALT text mô tả
- [ ] Có bảng so sánh dễ đọc trên mobile

---

## 8) Sau publish: Theo dõi & tối ưu

### Theo dõi chỉ số
- Impression / CTR / Position (GSC)
- Click vào URL đích (nếu có tracking)
- Tỷ lệ chuyển đổi từ bài commercial (nếu đo được)

### Quy tắc tối ưu
- **CTR thấp** → sửa Title/Meta (nhấn mạnh “so sánh theo ngân sách/nhu cầu”)
- **Traffic có nhưng ít click sang trang đích** → cải thiện CTA + vị trí internal link + block “gợi ý phù hợp”
- **Query mới phát sinh** → thêm section so sánh hoặc FAQ
